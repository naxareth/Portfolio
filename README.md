# Ace Philip Denulan | Portfolio

Welcome to the source code for my personal portfolio. I am a 4th-year BSIT student specializing in System Development and Applied Analytics, focused on building highly structured, aesthetically precise, and meticulously planned applications.

### 🌐 Live Site
**[https://acephilip.vercel.app/](https://acephilip.vercel.app/)**

## 🏗️ Architecture & Stack

- **Framework:** Next.js 16.2 (App Router)
- **Styling:** Tailwind CSS (Custom Dark/Light mode theme with specific typography and structural colors)
- **Animation:** Framer Motion (Orchestrating layout transitions, a cinematic global loading sequence, and scroll progress bars)
- **Deployment:** Vercel

## 📖 Design Philosophy

The aesthetic of this portfolio is heavily driven by a "System Builder" ethos:
- **Structure before speed:** Using blueprint-style subtle SVG watermarks (`WHATIDO-BG.svg`, `contact-footer-bg.svg`, etc.) to visually represent deep architectural planning.
- **Cinematic Execution:** High-impact initial load sequences, slow rotating background geometries, and heavily blurred image containers for elegant portrait displays.
- **Clean Contrast:** Relying on deep mochas, sharp charcoal dark modes, and soft sands to create a premium, editorial reading experience.

## 🚀 Getting Started

Clone this repository and install the dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
